### Dependensi
```bash

sudo apt-get install mpg123 dialog bc

# macOS

# Fedora/RHEL
```

### Buat script dapat di-eksekusi
chmod +x soundboard.sh

### Jalankan
./soundboard.sh

### Pindahkan ke path bin
sudo cp soundboard.sh /usr/local/bin/soundboard
